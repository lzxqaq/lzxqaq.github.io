本站架构

本站目前采用 Hugo（一个静态网站生成器）作为后台系统，使用 Github Pages 服务，源码托管在 Github 上。本站主题采用是 hugo-ivy（已经换成 hugo-xmin），作者 Yihui Xie。正是许多年前无意中进入作者的个人主页，被它的简洁之美吸引，我才开始了建站之旅（抄袭之旅）。

如果你也打算建立这样的博客网站，但是不知道从哪里开始，请看从零开始的博客建站之旅。
