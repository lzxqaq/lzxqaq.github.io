关于

我是谁

我出生于 1998 年，家在广东，毕业于华南师范大学软件学院，现在从事软件开发工作。我在广东长大，钟爱粤语歌。在运动爱好上，我喜欢打羽毛球，它也占据了我大部分的运动时间，另外，也喜欢跑步、篮球、骑行。喜欢看书，但是目前为止阅读量很少。我喜欢看一些陈年老番，比如《来自深渊》里那个名为“阿比斯”的深渊让人印象深刻；《红发少女安妮》，好喜欢那个哭着长大的女孩儿！

本站架构

本站目前采用 Hugo（一个静态网站生成器）作为后台系统，使用 Github Pages 服务，源码托管在 Github 上。本站主题采用是 hugo-ivy，作者 Yihui Xie。正是许多年前无意中进入作者的个人主页，被它的简洁之美吸引，我才开始了建站之旅（抄袭之旅）。

如果你也打算建立这样的博客网站，但是不知道从哪里开始，请看从零开始的博客建站之旅。
