本站架构

本站目前采用 Hugo（一个静态网站生成器）作为后台系统，使用 Github Pages 服务，源码托管在 Github 上。本站主题是 hugo-ivy 和 hugo-xmin 乱搭而成，作者 Yihui Xie。

如果你也打算建立这样的博客网站，但是不知道从哪里开始，请看从零开始的博客建站之旅。

另外，本网站托管在 Github上，所以可能打不开（ 多希望你能懂 ），多多多刷新一下或者换一个时间再来～
